# Deepak Chandru S — Portfolio & ATS Resume Website

A modern, dark-themed developer portfolio **and** ATS-friendly resume built with vanilla HTML, CSS, and JavaScript, showcasing the skills and experience of **Deepak Chandru S**, a Laravel Full-Stack Developer with 2.5+ years of experience.

---

## ✅ Completed Features

### 🏠 Portfolio — Hero Section
- Animated greeting with typewriter effect cycling through roles
- Interactive PHP code window showcasing developer profile
- Animated statistics counters (Years, Projects, Technologies)
- Call-to-action buttons (View My Work, View Resume)
- Scroll indicator with mouse animation

### 👤 Portfolio — About Me Section
- Animated avatar with pulsing gradient rings
- Technology badges (Laravel, PHP, MySQL, Server Mgmt)
- Detailed professional bio
- Quick info grid (Location, Degree, Experience, Languages)

### 🛠️ Portfolio — Skills Section
- 7 categorized skill cards:
  - Backend Development (Laravel v8–v11, PHP 8+, Composer, Eloquent ORM, Blade, Queues, Task Scheduling)
  - API Development (RESTful APIs, JSON, Versioning, Rate Limiting)
  - Authentication & Security (Passport, Sanctum, JWT, CSRF)
  - Database Management (MySQL, Query Optimization, Migrations, Seeders, Indexing)
  - Frontend Development (HTML5, CSS3, JavaScript, Bootstrap, Tailwind, jQuery, AJAX, Livewire)
  - DevOps & Hosting (Linux, cPanel/WHM, Nginx, Apache, Git, GitHub, SSH, SSL)
  - Tools & Platforms (VS Code, Postman, Figma, Laravel Excel, DomPDF, Razorpay, Firebase FCM, Twilio/MSG91)

### 💼 Portfolio — Experience Section
- Interactive timeline with glowing markers
- Detailed role description with key highlights
- Technology stack tags

### 🚀 Portfolio — Projects Section
- 6 project showcase cards:
  1. E-Commerce Platform
  2. API Backend for Mobile Apps
  3. Admin Dashboard & CMS
  4. Server Management & Deployment
  5. Notification & Communication System
  6. Authentication & Security Module

### 🎓 Portfolio — Education & Contact Sections
- Bachelor of Engineering (ECE) card with decorative elements
- Contact info cards (Email, Location, Availability)
- Social media links (GitHub, LinkedIn, Twitter, Stack Overflow)
- Interactive contact form with submission animation

### 🎨 Portfolio — UI/UX Features
- Dark theme with purple/teal gradient accents
- Particle background animation on canvas
- Cursor glow follow effect (desktop)
- Smooth scroll-based reveal animations (Intersection Observer)
- Responsive mobile navigation with hamburger menu
- Custom scrollbar styling
- Fully responsive design (desktop, tablet, mobile)

### 📄 ATS-Friendly Resume (NEW)
- **Clean, single-column layout** optimized for ATS parsing
- **Standard section headings**: Professional Summary, Core Competencies, Technical Skills, Professional Experience, Key Projects, Education, Certifications, Languages, Declaration
- **Print-optimized CSS** — Print/Save as PDF directly from browser
- **Download PDF button** in sticky action bar
- **No graphics/images** in print output for maximum ATS compatibility
- **Keyword-rich content** with all technologies, tools, and skills mentioned
- **16 detailed experience bullet points** covering all responsibilities
- **5 key project descriptions** with technology stacks
- **Skills table** organized by category for easy scanning
- Fully responsive on mobile/tablet
- Back to Portfolio navigation link
